# Keyboard shortcuts

## Gmail

### Inbox

| Action | key |
| ----- | ----- |
| Focus on toolbar | `,` |
| Select conversation | `x` |
| Go to inbox | `g + i` |
| Go to sent | `g + t` |
| Open conversation | `o` |
| Newer conversation | `k` |
| Older conversation | `j` |

### Conversation management

| Action | Key |
| ----- | ----- |
| Mute | `m` |
| Spam | `!` |
| Delete | `#` |
| Reply | `r` |
| Reply all | `a` |
| Forward | `f` |
| Mark read | `SHIFT + i` |
| Mark unread | `SHIFT + u` |
| Mark unread from selected | `_` |
| ----- | ----- |
| previous message in conversation | `p` |
| next message in conversation | `n` |

### Composing

| Action | Key |
| ----- | ----- |
| Send | `CTL + ENTER` |
| Add cc | `CTL + SHIFT + c` |
| Add bcc | `CTL + SHIFT + b` |
| Insert link | `CTL + k` |
| Remove formatting | `CTL + \` |

### Hangouts

| Action | Key |
| ----- | ----- |
| Show menu | `h + m` |
| Conversation list | `h + c `|
